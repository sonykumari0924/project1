Creating a New Repository Using Git (VS Code Terminal)

Step 1: Initialize a Repository

Use the following command to create a new Git repository:

git init

Step 2: Check Repository Status

To see the current status of your files:

git status

Step 3: Add Files

After creating your HTML and CSS files, add them to staging:

git add .

Step 4: Commit Changes

Save your changes with a commit message:

git commit -m "First commit"

Step 5: Push to GitHub (Initial Error)

If you try to push directly using:

git push origin main

You may get this error:


fatal: 'origin' does not appear to be a git repository
fatal: Could not read from remote repository.

This happens because no remote repository is connected yet.

Step 6: Create a Repository on GitHub

Create a new repository on GitHub.

Step 7: Add Remote Repository

Link your local repository to GitHub using:

git remote add origin <repository-link>

Step 8: Verify Remote Connection

Check if the remote repository is added correctly:

git remote -v

Step 9: Check Branch

Verify your current branch:

git branch

Step 10: Push Code to GitHub (First Time)

Use this command for the first push:

git push -u origin main

This sets the upstream branch, so next time you can simply use git push.

Step 11: Future Pushes

After setting upstream, you only need:

git push
